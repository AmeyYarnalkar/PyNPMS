

## What is Data?

Data is raw information collected for analysis.

Example:

* Age
* Salary
* Gender
* Date
* Sales amount
* Product category

But not all data is the same.

---

## Types of Data (The Foundation)

There are **two main categories**.

---

### A) Numerical Data (Quantitative)

Numbers that represent measurable quantities.

### 1. Discrete Data

* Countable
* Whole numbers
* Example: number of students, number of orders

You cannot have 2.5 students.

### 2. Continuous Data

* Measured
* Can have decimals
* Example: height, weight, salary, temperature

You can have 65.5 kg.

👉 Usually visualized with:

* Histogram
* Line plot
* Boxplot
* Scatter plot

---

### B) Categorical Data (Qualitative)

Represents categories or labels.

### 1. Nominal Data

* No order
* Example: gender, city, color

Male is not “greater” than female.

### 2. Ordinal Data

* Has order
* Example: low, medium, high
* Education level
* Rating (1–5 stars)

There is ranking here.

👉 Usually visualized with:

* Bar chart
* Count plot
* Pie chart (rarely recommended)

---

###  Special Data Types You Must Know

These matter in real-world datasets.

### 1. Time Series Data

* Data over time
* Example: daily sales
* Needs line plots

### 2. Boolean Data

* True / False
* Often treated as categorical

### 3. Text Data

* Needs preprocessing
* Not directly plottable

---

### Why Understanding Data Type is Critical

Because plotting wrong data type gives misleading graphs.

Example mistakes:

* Histogram for gender 
* Pie chart for continuous salary 
* Treating rating (1–5) as continuous without thinking 

If you don’t classify data first, your visualization becomes nonsense.

---

### Before Plotting — Do This Every Time

When you load a dataset:

```python
df.info()
df.describe()
df.head()
df.nunique()
```

Ask yourself:

* Which columns are numerical?
* Which are categorical?
* Any missing values?
* Any date columns?
* Any outliers?
